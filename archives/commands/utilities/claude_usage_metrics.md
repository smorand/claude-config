# Claude Usage Statistics Analysis

Execute comprehensive Claude Code usage analysis with cost tracking, performance metrics, and enterprise-grade reporting aligned with project project patterns.

## Pre-requisite
Check if ccusage is installed, else run pip install ccusage

## Analysis Framework

### 1. **Executive Cost Analysis**
Run `ccusage daily -b` command and provide structured financial analysis including:
- **Total Cost Breakdown**: Period totals with model-specific attribution
- **Performance Metrics**: Cache efficiency, token utilization, cost per operation
- **Trend Analysis**: Daily patterns, peak usage identification, cost optimization opportunities
- **Enterprise Reporting**: Professional format suitable for BTDP project documentation

### 2. **Key Performance Indicators (KPIs)**
Generate comprehensive metrics table with:
- **Token Efficiency**: Input/output ratios, cache read effectiveness
- **Cost Optimization**: Daily averages, peak analysis, trend identification
- **Model Performance**: Sonnet vs Opus utilization patterns
- **Project Impact**: Usage correlation with project development activities

### 3. **Operational Intelligence**
Provide actionable insights including:
- **Resource Optimization**: Cache strategy recommendations, model selection guidance
- **Cost Management**: Budget forecasting, efficiency improvements
- **Usage Patterns**: Development workflow optimization, peak period management
- **BTDP Compliance**: Enterprise reporting standards, cost center attribution

### 4. **Report Structure Requirements**

#### Executive Summary
- **Financial Overview**: Total costs, date range, usage days analysis
- **Performance Metrics**: Model-specific averages, peak usage analysis
- **Efficiency Indicators**: Cache effectiveness, cost per token analysis

#### Daily Cost Analysis
Focus on high-impact periods (top 15-20 days):
- **Date Analysis**: MM-DD format with model breakdown
- **Token Distribution**: Detailed usage patterns by model
- **Cost Attribution**: Daily totals with optimization insights

#### Strategic Insights
- **Usage Patterns**: High-cost day analysis ($20+ threshold)
- **Cache Optimization**: Effectiveness assessment with recommendations
- **Model Strategy**: Primary model identification and optimization
- **Trend Analysis**: Pattern recognition for future planning

#### Recommendations Framework
